# Vue + Vuetify: Firestore CRUD

![Vue.js](https://img.shields.io/badge/vuejs-%2335495e.svg?style=for-the-badge&logo=vuedotjs&logoColor=%234FC08D)![Vuetify](https://img.shields.io/badge/Vuetify-1867C0?style=for-the-badge&logo=vuetify&logoColor=AEDDFF)![Firebase](https://img.shields.io/badge/firebase-%23039BE5.svg?style=for-the-badge&logo=firebase)

Vue + Vuetify: A Simple Firestore CRUD Example Project, CRUD documents in collection using VueFire, a Vue Firestore Library

## How To Run

_Replace `pnpm` with your preferred package manager._

- Clone this repository `git clone https://github.com/bKoZii/vuetify-firestore-crud`
- Run `pnpm install` in this Project Folder.
- Run `pnpm dev` in this Project Folder.
